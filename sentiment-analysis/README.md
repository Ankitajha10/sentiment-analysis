# AI Sentiment Analyzer

A modern web application that uses Natural Language Processing (NLP) to detect the sentiment of a given text. Built with Flask and the Hugging Face Transformers library.

## 🚀 Features
- **Real-time Prediction**: Classifies text as Positive or Negative.
- **Lightweight Model**: Uses `DistilBERT` for high accuracy and fast inference.
- **Modern UI**: Clean, centered card layout with responsive design.
- **Interactive Feedback**: Loading spinners and dynamic color-coding (Green/Red).
- **Mobile Responsive**: Works perfectly on desktop, tablets, and phones.

## 🛠️ Technologies Used
- **Backend**: Python, Flask
- **Machine Learning**: Hugging Face Transformers, PyTorch
- **Frontend**: HTML5, CSS3, JavaScript (ES6)
- **Deployment**: Gunicorn, Render
